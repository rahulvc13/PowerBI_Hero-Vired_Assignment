# Agriculture Data Analysis Dashboard

## Project Overview
This dashboard analyzes rainfall patterns and crop yields across various Indian states to identify key agricultural trends.

## Key Insights
* **Top Performers:** Punjab and Tamil Nadu have the highest rice yields, much higher than the overall average.
* **Rainfall Impact:** Areas with moderate rainfall have more stable crop yields, while very high rainfall often causes variations.

## Recommendations
1. Try to gain knowledge or tips from highest rice yield states like Punjab and Tamil Nadu
2. Invest in micro-irrigation to maintain production during low-rainfall years.
